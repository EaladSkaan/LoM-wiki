# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="1010"></BookItemIcon>

飛石幫鎮幫秘笈，是幫主年輕時得高人傳授的獨門暗器手法。
<br><br>
據說練到精深處，可以在平靜水面上以飛石點水，彈跳八十八次。
