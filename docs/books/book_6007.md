# {{ $frontmatter.title }}

江湖第一美人奪魄幽蘭的成名絕技。
<br><br>
揉合奪魄門暗襲之術與雪山派內功，身法如鬼魅，能勾人魂魄。
<br><br>
摧心、蝕骨、銷魂、奪魄四式，一招狠似一招，極其毒辣，習之恐怕有傷天和。
