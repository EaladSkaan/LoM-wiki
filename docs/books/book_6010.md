# {{ $frontmatter.title }}

極度瘋狂的刀法，乃是一名身負奪妻之恨的三流刀客所創。
<br><br>
受冤下獄13年間，在心中揣想過幾千次敵人的武功路子，在腦海中將奪妻的仇敵凌遲幾萬次。
<br><br>
他出獄後不久，驚動江湖的血案「碧血狂殺」案件旋即爆發。
