# {{ $frontmatter.title }}

<BookItemIcon :size="`medium`" :needLink="false" :no="8101" :style="'float: right;'" />

藥王孫思邈所著，驗方經方俱備的醫學巨著。
<br><br>
此書基於《黃帝內經》臟腑學說，蒐羅醫論、醫方、用藥、針灸，共有二百三十門，藥方五千三百首。
<br><br>
開卷《大醫精誠》一文，首論醫德，仁懷天下，影響後世深遠。
<br clear="all" />
