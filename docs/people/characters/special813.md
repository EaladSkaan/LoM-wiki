# {{ $frontmatter.title }}

錦香宮有名的美人，臉看起來像掛軸的畫，實際上是化妝。
<br><br>
易容潛伏在唐門，叫華仙兒師姊，疑似和[[唐錚|二師兄]]有一腿，還會偷扁[[趙活]]。
<br><br>
是[[溫夫人]]的妹妹。武林大會錦香宮被圍攻時，與溫夫人演出一場戲度過危機，但[[龍湘]]一眼就看穿了她。

## 備註

- 疑似千面人魔
- 初至錦香宮時，二師兄好感越高，越容易骰到她導遊
- 嘴炮全人物第一
- 武功高強，[[趙活]]毫無還手之力，一刀死
