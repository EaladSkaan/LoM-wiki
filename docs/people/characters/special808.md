# 風神 解無塵

峨嵋派當代有名的年輕高手，人稱「風神」。  
與唐門飛俠唐布衣江湖齊名的峨嵋高手，擅長腿法。  
自詡為你大師兄的生平勁敵，非常煩人，見面就打架，也曾有結伴遊戲江湖的美談。

<Tabs>
  <Tab title="列傳一">
	解無塵雖是正派中人，但是為人好勝好戰，每逢成名高手必要糾纏挑戰，為此沒少受挫折。<br><br>
	獨門絕技《風雷神腳》乃是由家學《風神步》與峨嵋絕學《雷神腳》合一演變而來，施展開來可以邊踢邊借力，騰空踢到出人命。<br><br>
	「風神」的別稱是花了五兩銀子請人幫忙傳誦的，不然哪來這麼威風的外號。
