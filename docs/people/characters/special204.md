# {{ $frontmatter.title }}

崆峒派飛天掌門，有獨門仙法三昧真火。
<br><br>
二十多年前曾參加[青城論道](/event/past-青城論道)，曲解道籍使[趙逵](special403)道心沉淪，後日墮落成泥教地獄道法王，因此和[鄒博](special201)結了仇。
