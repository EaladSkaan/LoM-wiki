# 福韞

> 「蛤也一樣。施主，你問過魚和蝦的意願嗎？我猜沒有，你只想到你自己。」

嵩山派的年輕弟子，雲遊大宋，弘揚佛法，籌措善款以期修繕敗破的嵩山寺。

<Tabs>
  <Tab title="列傳一">
	天資聰穎，嘴力驚人，常在峨嵋、崆峒之間來回遊走，一邊弘揚佛法，一邊籌募善款修建嵩山寶剎。
	與良善之人相處抱持敬意，對頑劣之輩卻咄咄逼人。
	他對待的你的態度，取決於你的人品，一樣不免要嘮叨，差別在於用詞遣字氣不氣人。
