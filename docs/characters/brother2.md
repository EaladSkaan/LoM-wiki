# 二師兄 唐錚
## 列傳一
唐門掌門座下第二弟子。  
善使毒藥，毒學造詣尤勝乃師。  
為人心思縝密，孤僻深沉，律人律己皆嚴，受師命司掌戒律，深為同門所忌憚。  
平素等閒不出煉丹房，鮮少涉足江湖，故而名聲遠不及師兄，但與之交手過的對頭思及其人，盡皆不寒而慄。  
對於胡椒粉似乎有著異常的執著，非但自己喜歡，還會強迫大家一起吃。  

## 列傳二
::: warning
下有劇情捏他，點開前請注意。
:::
::: details
其真實身份是來自泥教人間道的臥底。  
授命於父，自小拜入唐門，以策應泥教。  
博得掌門信任，或傳「胡椒粉」秘方，本應以此挾制唐門弟子，供泥教驅策，卻因掌門廢止門人以胡椒粉毒練功的習俗而未能得逞。
:::
