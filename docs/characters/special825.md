# 夏靈犀

瑞笙的紅顏知己之一，苗疆巫靈教聖女。  
還在娘親腹中便被族中巫婆指作蛟龍妻。瑞笙與三五好友入苗疆，闖越五仙陣，揮劍斬大蛇，揭穿金蟾娘子和銀蛇公子的陰謀，喜聞樂見的英雄救美。
