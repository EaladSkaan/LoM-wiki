# {{ $frontmatter.title }}

鐵拳門「白眉虎」孟瞋，曾與飛天門[丹霞子](special404)，玄功門丹青羅剎[閻羅生](other2)，奪魂門[第三香](special820)，四人並稱崆峒四秀。練就鐵砂掌，有望角逐掌派人之位。
<br><br>
不幸得罪了[裂骨魔](girl6)，掌骨全都被捏爛了，醫好了也終生不能動武，淪落到早市去賣包子。
